# 语音记账

说一句话，AI 自动识别并完成记账。

## 使用方式

1. 在首页或记账页面长按麦克风图标
2. 说出记账内容
3. AI 自动识别并填充信息
4. 确认后保存

## 语音示例

```
"今天午餐花了35块"
→ 类型: 支出 | 分类: 餐饮 | 金额: 35 | 日期: 今天

"昨天打车回家花了28"
→ 类型: 支出 | 分类: 交通 | 金额: 28 | 日期: 昨天

"收到工资一万二"
→ 类型: 收入 | 分类: 工资 | 金额: 12000

"超市买菜花了156块8毛"
→ 类型: 支出 | 分类: 购物 | 金额: 156.8
```

## 支持的表达

- 金额：数字、中文数字、口语化表达
- 时间：今天、昨天、前天、上周等
- 分类：常见消费场景自动匹配

## 注意事项

- 需要麦克风权限
- 需要网络连接（使用云端 AI）
- 建议在安静环境使用

## 自动标签

使用语音记账时，系统会自动为交易添加「语音记账」标签，方便后续筛选和统计。
