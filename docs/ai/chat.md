# AI 对话

智能财务助手，用自然语言与 AI 交流。

## 进入对话

1. 点击「发现」→「AI 助手」
2. 或在首页点击 AI 图标

## 对话示例

### 记账

```
用户：帮我记一笔，今天中午吃饭花了 45 块
AI：好的，已为您记录一笔支出：
    - 金额：45 元
    - 分类：餐饮
    - 日期：今天
    是否确认保存？
```

### 查询

```
用户：这个月我花了多少钱？
AI：本月您的支出情况：
    - 总支出：3,256 元
    - 餐饮：1,200 元（36.8%）
    - 交通：450 元（13.8%）
    - ...
```

### 分析

```
用户：我的消费有什么问题吗？
AI：根据您的记账数据分析：
    1. 餐饮支出占比较高，建议...
    2. 娱乐消费集中在周末...
```

## 消息操作

长按对话中的消息，可以进行以下操作：

- **复制** - 复制消息内容
- **删除** - 删除单条消息

## 自定义提示词

在 AI 设置中可以自定义提示词，让 AI 更符合你的使用习惯。
