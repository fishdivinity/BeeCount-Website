# AI 助手概览

蜜蜂记账集成 AI 功能，让记账更智能、更轻松。

## 功能一览

| 功能 | 说明 |
|------|------|
| [语音记账](./voice) | 说一句话完成记账 |
| [图片识别](./image) | 拍照自动识别账单 |
| [AI 对话](./chat) | 智能财务助手 |

## 支持的 AI 服务

- **智谱 GLM** - 国内可用，推荐
- **本地模型** - 离线可用（开发中）

## 模型选择

智谱 GLM 提供多种模型可选：

| 模型 | 特点 | 推荐场景 |
|------|------|----------|
| GLM-4-Flash | 免费，速度快 | 日常记账，推荐 |
| GLM-4-Air | 平衡性能和成本 | 需要更好理解能力时 |
| GLM-4-AirX | 更强推理能力 | 复杂场景 |
| GLM-4-FlashX | 增强版免费模型 | 免费但需更好效果时 |
| GLM-4-Plus | 旗舰模型 | 追求最佳效果 |
| GLM-4-Long | 超长上下文 | 长对话场景 |

> 推荐使用 **GLM-4-Flash**，免费且满足日常记账需求。

## 配置 AI

1. 进入「我的」→「智能记账」→「AI 小助手」，或「发现」→「常用功能」→「AI 设置」
2. 选择 AI 服务提供商
3. 输入 API Key
4. 保存配置

## 获取 API Key

### 智谱 AI

1. 访问 [智谱开放平台](https://open.bigmodel.cn/)
2. 注册账号
3. 创建 API Key
4. 复制到蜜蜂记账

新用户通常有免费额度可用。
