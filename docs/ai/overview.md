# AI 助手概览

蜜蜂记账集成 AI 功能，让记账更智能、更轻松。

## 功能一览

| 功能 | 说明 |
|------|------|
| [语音记账](./voice) | 说一句话完成记账 |
| [图片识别](./image) | 拍照自动识别账单 |
| [AI 对话](./chat) | 智能财务助手 |

## 支持的 AI 服务

- **智谱 GLM** - 国内可用，推荐
- **本地模型** - 离线可用（开发中）

## 配置 AI

1. 进入「我的」→「智能记账」→「AI 小助手」，或「发现」→「常用功能」→「AI 设置」
2. 选择 AI 服务提供商
3. 输入 API Key
4. 保存配置

## 获取 API Key

### 智谱 AI

1. 访问 [智谱开放平台](https://open.bigmodel.cn/)
2. 注册账号
3. 创建 API Key
4. 复制到蜜蜂记账

新用户通常有免费额度可用。
