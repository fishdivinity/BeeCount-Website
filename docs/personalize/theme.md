# 主题色设置

蜜蜂记账支持自定义主题色，打造个性化界面。

![个性化设置](/img/preview/zh/10-personalization.png)

## 修改主题色

1. 进入「我的」→「外观设置」→「个性装扮」
2. 点击「主题色」
3. 选择预设颜色或自定义

## 预设颜色

- 🟡 **蜜蜂黄** - 默认主题色
- 🔵 **天空蓝**
- 🟢 **森林绿**
- 🟣 **优雅紫**
- 🔴 **热情红**
- 🟠 **活力橙**

## 自定义颜色

点击调色板图标，可以选择任意颜色作为主题色。

## 主题色应用

主题色会应用到：

- 导航栏
- 按钮
- 选中状态
- 图标强调色
- 图表配色

## 字体大小

支持调整应用字体大小：

- 小
- 标准（默认）
- 大
- 超大

适合不同视力需求的用户。
