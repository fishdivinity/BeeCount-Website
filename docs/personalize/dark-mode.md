# 暗黑模式

蜜蜂记账支持暗黑模式，护眼且省电。

## 开启暗黑模式

1. 进入「我的」→「外观设置」→「个性装扮」
2. 点击「深色模式」
3. 选择模式：
   - **跟随系统** - 自动跟随系统设置
   - **浅色模式** - 始终使用亮色
   - **深色模式** - 始终使用暗黑

## 设计特点

蜜蜂记账的暗黑模式采用 **纯黑背景 + 主题色边框** 设计：

- ⚫ **纯黑背景** - OLED 屏幕省电
- 🟡 **主题色边框** - 保留个性化主题色
- 👁️ **护眼** - 降低屏幕亮度刺激

## 全面适配

暗黑模式下，所有界面元素都已适配：

- 所有页面
- 弹窗对话框
- 数字键盘
- 图表组件
- 系统控件

## OLED 友好

纯黑背景在 OLED 屏幕上：

- 像素关闭，真正省电
- 对比度更高
- 夜间使用更舒适
