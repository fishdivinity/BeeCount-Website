# 二级分类

蜜蜂记账支持父子分类结构，让分类更精细。

## 什么是二级分类？

二级分类是父分类下的子分类，例如：

```
餐饮（父分类）
├── 早餐
├── 午餐
├── 晚餐
└── 外卖

交通（父分类）
├── 公交地铁
├── 打车
├── 加油
└── 停车费
```

## 创建二级分类

1. 进入「我的」→「数据管理」→「分类管理」
2. 点击「+」添加分类
3. 在「父分类」中选择一个已有分类
4. 输入子分类名称
5. 保存

## 使用二级分类

记账时：

1. 先选择父分类
2. 再选择子分类（如有）

## 统计分析

- 可以按父分类汇总查看
- 也可以展开查看子分类详情

## 显示模式

在分类管理中可以切换：

- **层级模式** - 显示父子结构
- **平铺模式** - 所有分类平铺显示
