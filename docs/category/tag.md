# 标签系统

标签是比分类更灵活的归类方式，一笔交易可以打多个标签。

## 标签 vs 分类

| 特性 | 分类 | 标签 |
|------|------|------|
| 数量 | 每笔只能一个 | 可以多个 |
| 结构 | 支持层级 | 扁平结构 |
| 用途 | 固定分类 | 灵活标记 |

## 使用场景

- **报销** - 标记需要报销的交易
- **旅行** - 某次旅行的所有消费
- **项目** - 某个项目相关支出
- **礼物** - 送礼或收礼记录
- **临时标记** - 任何需要临时归类的场景

## 创建标签

1. 进入「我的」→「数据管理」→「标签管理」
2. 点击「+」
3. 输入名称，选择颜色
4. 保存

## 使用标签

记账时点击「添加标签」，可选择一个或多个标签。

## 标签统计

点击标签查看详情：

- 关联交易数量
- 总金额统计
- 交易列表

适合查看某次旅行总花费、某个项目总支出等。
