# 交易附件

为交易添加图片附件，保存消费凭证、发票等。

## 添加附件

### 记账时添加

1. 在记账页面，点击底部的「附件」图标
2. 选择「拍照」或「从相册选择」
3. 图片会自动添加为附件
4. 完成记账后，附件随交易一起保存

### 编辑时添加

1. 打开已有交易的详情页
2. 点击「编辑」
3. 在附件区域添加图片
4. 保存修改

## 查看附件

- 在交易列表中，有附件的交易会显示附件图标
- 点击交易进入详情页，可以查看所有附件
- 点击附件图片可放大查看

## 删除附件

1. 进入交易编辑页面
2. 长按要删除的附件图片
3. 确认删除

## 自动保存附件

使用以下方式记账时，图片会自动保存为附件：

- **图片记账** - 识别用的图片自动保存
- **拍照记账** - 拍摄的照片自动保存

## 附件导出

导出交易数据时，可以选择是否包含附件：

1. 进入「我的」→「数据管理」→「导出数据」
2. 选择「包含附件」选项
3. 导出的压缩包中会包含附件文件夹

## 注意事项

- 附件存储在本地，会占用设备存储空间
- 云同步时，附件也会同步（取决于云存储方案）
- 建议定期清理不需要的附件
