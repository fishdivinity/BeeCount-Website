# iOS 自动记账

通过 iOS 快捷指令 + 截图识别，实现支付后快速记账。

## 工作原理

1. 创建快捷指令：截屏 → 蜜蜂记账识别
2. 绑定到「轻点背面」手势
3. 支付完成后双击手机背部
4. 自动截图并识别金额，完成记账

## 配置步骤

### 方式一：App 内引导（推荐）

1. 打开蜜蜂记账，进入「发现」→「快捷指令」
2. 按照引导页面的步骤操作
3. 点击「添加快捷指令」自动创建
4. 根据提示绑定到轻点背面

### 方式二：手动创建

#### 1. 创建快捷指令

1. 打开「快捷指令」应用，点击右上角「+」创建新快捷指令
2. 添加「截屏」操作
3. 搜索并添加「蜜蜂记账 - 截图自动记账」操作
4. 将「蜜蜂记账」的截图参数设置为上一步的「截屏」
5. 保存快捷指令

#### 2. 绑定轻点背面（推荐）

1. 打开「设置」>「辅助功能」>「触控」>「轻点背面」
2. 选择「轻点两下」或「轻点三下」
3. 选择刚创建的快捷指令
4. 完成！支付时双击手机背部即可快速记账

> ✅ 推荐：在「轻点背面」中绑定快捷指令后，支付时双击手机背部即可自动截图并识别记账，无需手动截图。

## 使用方式

1. 完成支付后，停留在支付成功页面
2. 双击手机背部（或手动运行快捷指令）
3. 自动截图并识别金额、商家信息
4. 确认后完成记账

## 其他触发方式

除了轻点背面，还可以通过以下方式触发：

- 对 Siri 说快捷指令名称
- 从快捷指令 Widget 点击
- 添加到主屏幕作为图标

## 常见问题

**轻点背面没反应？**

1. 确认 iPhone 型号支持（iPhone 8 及以上）
2. 检查是否正确绑定了快捷指令
3. 尝试调整轻点力度和位置

**识别不准确？**

1. 确保截图包含完整的支付金额
2. 支付成功页面停留时间要足够
3. 可以在蜜蜂记账中手动调整
