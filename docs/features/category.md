# 分类管理

蜜蜂记账支持自定义分类，并支持二级分类结构。

## 预设分类

应用内置了常用的收支分类：

**支出分类**：餐饮、交通、购物、娱乐、居住、通讯、医疗等

**收入分类**：工资、奖金、投资收益、兼职等

## 自定义分类

1. 进入「我的」→「数据管理」→「分类管理」
2. 点击右上角「+」添加新分类
3. 选择图标、名称和颜色
4. 可选择父分类创建二级分类

## 二级分类

支持创建父子分类结构，如：

- 餐饮
  - 早餐
  - 午餐
  - 晚餐
  - 外卖

## 分类排序

长按分类可以拖拽排序，将常用分类放在前面。

## 分类迁移

删除分类时，可以将该分类下的交易迁移到其他分类。
