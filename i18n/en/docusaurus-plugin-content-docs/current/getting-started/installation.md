# Installation Guide

BeeCount is available for both iOS and Android. Choose the installation method that works best for you.

## iOS Installation

### Option 1: App Store (Recommended)

1. Open the App Store
2. Search for "BeeCount - Simple Expense Tracker" or "BeeCount"
3. Tap "Get" to download and install

> Note: The App Store version is currently pending approval in mainland China. Available in all other regions.

[Go to App Store](https://apps.apple.com/app/id6754611670)

### Option 2: TestFlight

TestFlight is Apple's official beta testing platform, available worldwide.

1. First, install [TestFlight](https://apps.apple.com/app/testflight/id899247664)
2. Open the test link: https://testflight.apple.com/join/Eaw2rWxa
3. Tap "Accept" to join the beta
4. Install BeeCount from TestFlight

[Join TestFlight](https://testflight.apple.com/join/Eaw2rWxa)

## Android Installation

### GitHub Release

1. Visit the [GitHub Release page](https://github.com/TNT-Likely/BeeCount/releases/latest)
2. Download the latest APK file
3. Open the APK to install
4. If prompted about "Unknown sources", allow installation in settings

[Download APK](https://github.com/TNT-Likely/BeeCount/releases/latest)

### Mirror Acceleration (China)

If GitHub downloads are slow, you can use mirror acceleration:

1. When a new version is detected in the app, tap "Update"
2. If download is slow, tap the "Mirror Acceleration" button
3. Select an available mirror source (with speed test)
4. Download the update quickly via the mirror

Supported mirrors:
- ghproxy.net
- ghproxy.cc
- gh-proxy.com
- ghfast.top
- gh.noki.icu
- github.moeyy.xyz
