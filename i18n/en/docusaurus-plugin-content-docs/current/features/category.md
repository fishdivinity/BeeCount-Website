# Category Management

BeeCount supports custom categories with a two-level hierarchy structure.

## Preset Categories

The app comes with commonly used income and expense categories:

**Expense Categories**: Food, Transportation, Shopping, Entertainment, Housing, Communication, Medical, etc.

**Income Categories**: Salary, Bonus, Investment Returns, Side Income, etc.

## Custom Categories

1. Go to "Me" → "Data Management" → "Category Management"
2. Tap "+" in the top right to add a new category
3. Select an icon, name, and color
4. Optionally select a parent category to create a subcategory

## Subcategories

Create parent-child category structures like:

- Food
  - Breakfast
  - Lunch
  - Dinner
  - Takeout

## Category Sorting

Long-press and drag categories to reorder them, placing frequently used ones at the top.

## Category Migration

When deleting a category, you can migrate its transactions to another category.
