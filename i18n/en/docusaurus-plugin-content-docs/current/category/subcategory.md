# Subcategories

BeeCount supports parent-child category structures for more detailed organization.

## What are Subcategories?

Subcategories are children of parent categories, for example:

```
Food (parent category)
├── Breakfast
├── Lunch
├── Dinner
└── Takeout

Transportation (parent category)
├── Public Transit
├── Taxi/Rideshare
├── Gas
└── Parking
```

## Create a Subcategory

1. Go to "Me" → "Data Management" → "Category Management"
2. Tap "+" to add a category
3. In "Parent Category", select an existing category
4. Enter the subcategory name
5. Save

## Using Subcategories

When recording:

1. First select the parent category
2. Then select the subcategory (if available)

## Statistics & Analysis

- View summaries grouped by parent category
- Expand to see subcategory details

## Display Mode

In category management, you can switch between:

- **Hierarchy Mode** - Shows parent-child structure
- **Flat Mode** - Shows all categories in a flat list
