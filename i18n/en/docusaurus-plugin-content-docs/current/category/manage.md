# Category Management

Categories help you organize income and expenses for easy analysis.

![Category Management](/img/preview/zh/09-category-management.png)

## Preset Categories

The app includes commonly used categories:

**Expenses**: Food, Transportation, Shopping, Entertainment, Housing, Communication, Medical, Education...

**Income**: Salary, Bonus, Investment Returns, Side Income, Red Packets...

## Add a Category

1. Go to "Me" → "Data Management" → "Category Management"
2. Tap "+" in the top right
3. Select an icon and color
4. Enter category name
5. Save

## Edit a Category

Tap a category to open the edit page where you can modify:

- Category name
- Icon
- Color

## Sort Categories

Long-press and drag categories to reorder them. Place frequently used categories at the top for faster recording.

## Delete a Category

When deleting a category, you can choose to migrate its transactions to another category.

![Category Migration](/img/preview/zh/08-category-migration.png)
