# Tag System

Tags offer more flexibility than categories - a single transaction can have multiple tags.

## Tags vs Categories

| Feature | Category | Tag |
|---------|----------|-----|
| Quantity | One per transaction | Multiple allowed |
| Structure | Hierarchical | Flat |
| Purpose | Fixed classification | Flexible labeling |

## Use Cases

- **Reimbursement** - Mark transactions that need reimbursement
- **Travel** - All expenses from a specific trip
- **Project** - Expenses related to a specific project
- **Gift** - Giving or receiving gifts
- **Temporary Marking** - Any scenario needing temporary grouping

## Create a Tag

1. Go to "Me" → "Data Management" → "Tag Management"
2. Tap "+"
3. Enter name, select color
4. Save

## Using Tags

When recording, tap "Add Tag" to select one or more tags.

## Tag Statistics

Tap a tag to view details:

- Number of associated transactions
- Total amount
- Transaction list

Great for viewing total spending on a trip or total expenses for a project.
